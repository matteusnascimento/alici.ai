📋 Especificações de Animação

Estados

Idle: Respiração suave, piscadas ocasionais, postura neutra
Thinking: Símbolos matemáticos orbitando, expressão contemplativa
Talking: Boca animada, movimento fluido, comunicativo
Happy: Sparkles, sorriso radiante, movimento energético
Serious: Raios de alerta, expressão tensa, aura vermelha
Mystical: Runas cósmicas, levitação, atmosfera espiritual

Paleta de Cores

Ciano
Azul
Roxo
Âmbar
Vermelho
Verde

Características Visuais

Cabelo: Ciano/azul claro, comprido, fluido com ondulações suaves
Olhos: Azuis brilhantes com reflexo cyberpunk, pupila dinâmica
Aura: Circular pulsante, muda cor com emoção
Partículas: Símbolos matemáticos, sparkles, runas cósmicas
Efeitos: Glow, brilho, distorção espiritual (quando mystical)

Velocidades de Animação

Muito Lenta: 0.04 (Mystical)
Lenta: 0.08 (Thinking, Idle)
Normal: 0.12 (Talking, Default)
Rápida: 0.18 (Happy, Energético)
Muito Rápida: 0.25 (Efeitos especiais)

📚 Uso da API

<canvas id="alici-canvas"></canvas>

<script src="alici_renderer.js"></script>
<script>
  const controller = new AliciAnimationController("alici-canvas");

  // Mudar estado
  controller.setState("talking", "happy", 0.18);

  // Opções de estado
  // "idle" | "thinking" | "talking" | "happy" | "serious" | "mystical"

  // Opções de emoção
  // "neutral" | "thinking" | "happy" | "serious" | "mysterious"

  // Parar animação
  controller.stop();
</script>
