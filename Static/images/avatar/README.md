# Avatar Images

Este diretório contém as imagens do avatar holográfico da Alici.

## Imagens necessárias:

1. **alici_idle.jpg** - Avatar em estado idle (respirando)
2. **alici_speaking.jpg** - Avatar falando (movimento de boca)
3. **alici_listening.jpg** - Avatar ouvindo
4. **alici_thinking.jpg** - Avatar pensando

## Especificações recomendadas:

- **Formato**: JPG ou PNG
- **Dimensões**: 400x400px (mínimo) ou maior com aspect ratio 1:1
- **Estilo**: Cyberpunk/futurista com cores neon compatíveis com o tema holográfico
- **Cores principais**: Ciano (#00ffff), verde (#00ffaa), magenta (#ff00ff), dourado (#ffd700)

## Nota:

Atualmente, o avatar usa um placeholder (◆) com animações holográficas.
Quando você adicionar imagens reais aqui, elas serão automaticamente carregadas pela interface.

Para adicionar suas imagens:
1. Coloque os arquivos JPG/PNG neste diretório
2. Nomeie-os exatamente como listado acima
3. A interface carregará automaticamente as imagens
