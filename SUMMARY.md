## 🏗️ Architecture Overview

### 6-Layer Decision Engine
1. **Identity** - Fixed responses for "who are you"
2. **Memory** - PostgreSQL persistent learning
3. **Local Rules** - 80+ pattern-based responses
4. **Neural Models** - TensorFlow .h5 inference
5. **Web Search** - DuckDuckGo integration
6. **Fallback** - Graceful degradation

### Technology Stack
- **Backend:** FastAPI + Python 3.11
- **Database:** PostgreSQL (Neon.tech)
- **Auth:** JWT with bcrypt
- **ML:** TensorFlow/Keras
- **Search:** DuckDuckGo API
