## ⚡ Troubleshooting

Se algo não funcionar:

### Servidor offline?
```bash
# Reinicie em uma terminal nova:
cd c:\Users\PC\Videos\alici.ai
uvicorn alici_api.app:create_app --host 127.0.0.1 --port 8001
```

### Porta ocupada?
```bash
# Use outra porta:
uvicorn alici_api.app:create_app --host 127.0.0.1 --port 8002
```

### Quer verificar tudo novamente?
```bash
python verify_platform_detailed.py
```
